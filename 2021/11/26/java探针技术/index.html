vm参数

  • -javaagent:[=<选项>]
    加载本机代理库 , 例如 -agentlib:hprof
  • -agentlib:[=<选项>]
    按完整路径名加载本机代理库
  • -agentpath:[=<选项>]
    加载 Java 编程语言代理, 请参阅 java.lang.instrument
